Programy

Wtyczka programy

Wtyczka Programy dostarcza listę wszystkich zainstalowanych programów.

Podstawy obsługi

Uruchamianie programu

  1. Aktywacja Kupfera i wpisanie skrótu nazwy programu.

  2. Naciśnięcie Return w celu uruchomienia programu.

Otwieranie pliku z użyciem określonej aplikacji.

  1. Wybranie pliku w Kupferze

  2. Naciśnięcie Tab, wybór akcji Otwórz za pomocą.

  3. Type Tab to select the third (indirect object) pane and in it, select the application to open the file with.

The action Open opens files and folders with the registered default application for that specific file type. This can be configured using the action Set Default Application....

The file type associations are provided by the desktop environment, and modifying them here will modify them on the desktop as well. This may not be true when not using GNOME or XFCE.

Selecting default application for a file type or folders

  1. Select a file or folder in Kupfer

  2. Type Tab and select the action Set Default Application...

  3. Type Tab to select the third (indirect object) pane and in it, select the application to always open the file type with.

Konfiguracja

Parametr konfiguracyjny Programy środowiska graficznego w panelu informacyjnym wtyczki w Preferencjach Kupfera określa jak Kupfer powinien zachowywać się przy pobieraniu konfiguracji środowiska graficznego - wiele aplikacji nie powinno być pokazywanych w zależności od aktualnie używanego środowiska graficznego. Domyślnym ustawieniem jest środowisko GNOME.
