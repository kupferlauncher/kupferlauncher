Klawisze poleceń oraz klawisze akceleratorów

Klawiatura

W trybie poleceń następujące klawisze posiadają specjalne znaczenie:

↓ lub Spacja

Przejście do następnego znalezionego obiektu

↑ lub Shift+Spacja

Przejście do poprzedniego znalezionego elementu

→ lub /

Wejście do obiektu posiadającego zawartość

Backspace

Usunięcie znaku z zapytania. Jeżeli zapytanie jest puste - przejście na wyższy poziom.

.

Aktywacja trybu "tekstowego"

,

Odłożenie zaznaczonego obiektu na stosie ("trik z przecinkiem")

Klawisze szybkiego dostępu:

=

Aktywacja trybu tekstowego z przedrostkiem = (dla Kalkulator)

/

Aktywacja trybu tekstowego (gdy nic nie jest zaznaczone) z prefiksem / (dla wprowadzania ścieżki względem korzenia /).

Dodatkowo:

  • Return aktywuje zaznaczony obiekt - polecenie jest wykonywane.

  • Escape usuwa aktualne zaznaczenie.

  • Tab przełącza między obiektem i panelem akcji.

Znaczenie klawiszy Spacja, ., ,, / oraz = może być zmienione, ale w tym celu należy najpierw wyłączyć ich specjalne działanie przez odznaczenie pola Włączenie jedno-klawiszowych poleceń w Preferencjach Kupfera.

Konfiguracja skrótów klawiszowych

  1. Otwarcie Preferencji Kupfera i przejście do karty Klawiatura.

  2. Dwukrotne kliknięcie wiersza ze skrótem pod Skróty klawiszowe listy elementów.

  3. Za pomocą okna, które zostanie wyświetlone, nacisnąć klawisz modyfikatora i klawisz litery lub cyfry w celu ustawienia skrótu. Przykładowo naciskając i przytrzymując klawisz Ctrl oraz naciskając i puszczając klawisz T zostanie przypisany skrót Ctrl+T.

Action Accelerators

Action acclerators are a quick way to select or run a certain action on an object. All action accelerators use the same modifier key, which is Ctrl by default.

An action's accelerator is a single letter that is pressed together with the modifier to run the action (in the case of actions that need an indirect object, the third pane will be selected instead).

How to configure an action accelerator

  1. Open Kupfer and find an object that allows the action you are configuring. Switch to the action pane with Tab.

  2. Find the action to configure an accelerator for in the action pane.

  3. Click the kupfer context menu in the window and select the option Assign Accelerator to “(Action)”

  4. Type a single letter (without modifier) in the window that pops up. Escape cancels the configuration, and Backspace unsets the accelerator key for the action.

Regular keyboard shortcuts in Kupfer take precedence over action accelerator keys. If multiple simultaneously eligible actions have the same accelerator key, the highest ranked one will be activated.

Globalne skróty klawiszowe

Kupfer zawsze reaguje na globalne skróty klawiszowe, nawet gdy nie jest w danej chwili aktywny.

Globalne skróty klawiszowe obejmują następujące akcje i posiadają domyślne przypisania:

Ctrl+Spacja

Pokaż / ukryj Kupfera

(domyślnie nieskonfigurowane)

Pokazanie Kupfera z wybranym obiektem "Zaznaczenie".

Konfiguracja globalnych skrótów klawiszowych

  1. Otwarcie Preferencji Kupfera i przejście do karty Klawiatura.

  2. Dwukrotne kliknięcie wiersza skrótu pod Globalne skróty klawiszowe.

  3. Za pomocą okna, które zostanie wyświetlone, nacisnąć klawisz modyfikatora i klawisz litery lub cyfry w celu ustawienia skrótu. Przykładowo naciskając i przytrzymując klawisz Super oraz naciskając i puszczając klawisz Spacja zostanie przypisany skrót Super+Spacja.

Skróty klawiszowe są globalne co oznacza, że mogą być użyte z dowolnego programu. Należy uważnie wybierać skróty klawiszowe, tak aby nie spowodowały one problemów z innymi funkcjami.

Wyzwalacze

Wtyczka Wyzwalacze pozwala na aktywację akcji za pomocą globalnych skrótów klawiszowych. Skróty klawiszowe wtyczki wyzwalaczy nie są wyświetlane w oknie Preferencji Kupfera.

Więcej informacji można znaleźć w Wyzwalacze.
