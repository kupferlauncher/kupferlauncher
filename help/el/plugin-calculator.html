Αριθμομηχανή

Πρόσθετο αριθμομηχανής

Το πρόσθετο αριθμομηχανής σας επιτρέπει να υπολογίσετε γρήγορα παραστάσεις. Μπορεί να αξιολογεί παραστάσεις που καταχωρούνται ως κείμενο αρχίζοντας με "=". Εισάγοντας = από την κατάσταση εντολών θα ξεκινήσει η λειτουργία κειμένου απευθείας με πρόθεμα = για γρήγορη πρόσβαση.

Βασική χρήση

  1. Ενεργοποιήστε το Kupfer και πληκτρολογήστε =

  2. Πληκτρολογήστε μια μαθηματική παράσταση χρησιμοποιώντας τα +,-,/,* (και ** για εκθέτη)

  3. Πατήστε το Επιστροφή για να λάβετε το αποτέλεσμα.

Σημειώσεις

Η Αριθμομηχανή χρησιμοποιεί τις ενότητες μιγαδικών μαθηματικών και python και αναλύει τις παραστάσεις ως παραστάσεις Python. Μπορείτε να χρησιμοποιήσετε κοινές μαθηματικές συναρτήσεις, όπως sqrt, sin, exp και log· η εντολή =help θα εμφανίσει μια λίστα με όλες τις προσδιορισμένες συναρτήσεις και σταθερές.

  • Σημειώστε ότι ο τελεστής δύναμης στο Python είναι διπλά αστέρια, για παράδειγμα =3**3 θα αξιολογηθεί σε 27.

  • Για να υπολογίσετε τριγωνομετρικές συναρτήσεις για τις γωνίες, μετατρέψετε πρώτα σε ακτίνια:

    sin(radians(30)) -> 0.5

  • Το τελευταίο αποτέλεσμα αποθηκεύεται ως όνομα _ (ένα χαρακτήρα υπογράμμισης, όπως ακριβώς και στην κονσόλα Python).
