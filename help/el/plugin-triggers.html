Εναύσματα

Προσθετο Εναύσματα

Με τα Εναύσματα μπορείτε να πάρετε μια εντολή που θα εκτελούσατε κανονικά στο Kupfer, όπως την εκκίνηση μιας εφαρμογής ή το άνοιγμα ενός εγγράφου, και να δεσμεύσετε ένα γενικό συνδυασμό πλήκτρων για να εκτελέσετε αυτήν την εντολή. Για όσο το Kupfer εκτελείται, οποιαδήποτε εφαρμογή μπορεί να έχει την τρέχουσα εστίαση.

Βασική χρήση

Δημιουργία ενός νέου εναύσματος

  1. Ρύθμιση μιας εντολής στο Kupfer που θέλετε να εκτελέσετε

  2. Πατήστε Ctrl+Επιστροφή για να δημιουργήσετε μια Εντολή σύνθεσης.

  3. Πατήστε Tab και επιλέξτε την ενέργεια Προσθήκη εναύσματος... και πιέστε Επιστροφή

  4. Χρησιμοποιώντας το παράθυρο που εμφανίζεται, πατήστε τα πλήκτρα τροποποίησης και ένα γράμμα ή αριθμό για να ορίσετε μια γενική συντόμευση. Για παράδειγμα πατώντας και κρατώντας πατημένα τα Ctrl+Alt και, στη συνέχεια, πιέζοντας και αφήνοντας το R δεσμεύονται τα Ctrl+Alt+R.

Αφαίρεση εναύσματος

  1. Βρείτε τα πηγαία Εναύσματα στο Kupfer και πιέστε → για να δείτε τα περιεχόμενά του.

  2. Βρείτε το ενεργό έναυσμα στον κατάλογο και χρησιμοποιήσετε την ενέργεια Αφαίρεση εναύσματος.

Δεδομένου ότι οι συντομεύσεις πληκτρολογίου των εναυσμάτων είναι γενικές, μπορούν να χρησιμοποιηθούν από οποιαδήποτε εφαρμογή. Επιλέξτε συντομεύσεις με προσοχή έτσι ώστε να μην έρχονται σε διένεξη με άλλες λειτουργίες.

Συμβουλές για έμπειρους χρήστες

Proxy objects such as Selected Text, Clipboard Text, Clipboard File, Frontmost Window, Last Command et cetera allow very powerful triggers.

Η χρήση της εφαρμογής Εκκίνηση είναι ειδική: εαν δευσμεύσετε ένα έναυσμα με την Εκκίνηση, θα ξεκινήσει η εφαρμογή, εάν δεν εκτελείται, αλλά θα εστιάσει στην εφαρμογή όταν εκτελείται ήδη. Αντίθετα, η Έναρξη εκ νέου θα ξεκινήσει την εφαρμογή είτε βρίσκεται σε λειτουργία είτε όχι.

Παράδειγμα εναυσμάτων

Επεξεργαστής κειμένου → Εκκίνηση

Ξεκινήστε τον Επεξεργαστή κειμένου ή εστιάστε στο παράθυρό του, εάν είναι ήδη σε λειτουργία.

Selected Text → Search in Wikpedia

Search using the currently selected text.

Τραγούδια → Αναζήτηση περιεχομένων

Μπορείτε να προσθέσετε ακόμη και ένα έναυσμα για να ανοίξει ένα συγκεκριμένο υποκατάλογο στο Kupfer, για παράδειγμα, τα τραγούδια του Rhythmbox.
