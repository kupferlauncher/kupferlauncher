Εφαρμογές

Πρόσθετο εφαρμογών

Το Εφαρμογές παρέχει όλα τα εγκατεστημένα προγράμματα.

Βασική χρήση

Εκκίνηση μιας εφαρμογής

  1. Ενεργοποιήστε το Kupfer και πληκτρολογήστε μια συντομογραφία του ονόματος εφαρμογής

  2. Πατήστε Επιστροφή για να ξεκινήσει η εφαρμογή.

Άνοιγμα ενός αρχείου με μια συγκεκριμένη εφαρμογή

  1. Επιλέξτε ένα αρχείο στο Kupfer

  2. Πληκτρολογήστε Tab και επιλέξτε την ενέργεια Άνοιγμα με ...

  3. Πληκτρολογήστε Tab για να επιλέξετε το τρίτο παράθυρο (έμμεσο αντικείμενο) και σε αυτό, επιλέξτε την εφαρμογή με την οποία να ανοίξει το αρχείο.

Η ενέργεια Άνοιγμα ανοίγει τα αρχεία και τους φακέλους με την εγγεγραμμένη προεπιλεγμένη εφαρμογή για το συγκεκριμένο τύπο αρχείου. Αυτό μπορεί να ρυθμιστεί χρησιμοποιώντας την ενέργεια Ορισμός προεπιλεγμένης εφαρμογής....

Οι συσχετισμοί του τύπου αρχείου παρέχονται από το περιβάλλον εργασίας, και η τροποποίηση τους εδώ θα τους τροποποιήσει επίσης και στην επιφάνεια εργασίας. Αυτό δεν μπορεί να είναι εφικτό όταν δεν χρησιμοποιείτε το GNOME ή το XFCE.

Επιλέγοντας την προεπιλεγμένη εφαρμογή για έναν τύπο αρχείου ή φακέλων

  1. Επιλέξτε ένα αρχείο ή φάκελο στο Kupfer

  2. Πληκτρολογήστε Tab και επιλέξτε την ενέργεια Ορισμός προεπιλεγμένης εφαρμογής...

  3. Πληκτρολογήστε Tab για να επιλέξετε το τρίτο παράθυρο (έμμεσο αντικείμενο) και σε αυτό, επιλέξτε την εφαρμογή με την οποία θα ανοίγει πάντα ο τύπος του αρχείου.

Διαμόρφωση

The configuration option Applications for Desktop Environment in the plugin's information pane in Kupfer Preferences determines how Kupfer should act with regard to the desktop environment configuration—certain applications request not to be shown depending on which desktop environment is used.

The default setting is blank which means that it will use the environment variable XDG_CURRENT_DESKTOP to identify the desktop environment automatically.
