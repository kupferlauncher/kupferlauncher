Πλήκτρο backspace

Διαγραφή ενός χαρακτήρα από το ερώτημα. Εάν το ερώτημα είναι άδειο, ανεβαίνει ένα επίπεδο προς τα πάνω

.

Ενεργοποίηση λειτουργίας ελεύθερου κειμένου

,

Τοποθετεί το επιλεγμένο αντικείμενο στη στοίβα ("Τέχνασμα κόμμα")

Πλήκτρα γρήγορης πρόσβασης:

=

Ενεργοποίηση λειτουργίας ελεύθερου κειμένου με πρόθεμα = (για το Αριθμομηχανή)

/

Ενεργοποίηση λειτουργίας ελεύθερου κειμένου (όταν δεν έχει επιλεγεί τίποτα) με πρόθεμα / (για την είσοδο μιας ριζωμένης διαδρομής)

Επιπλέον:

  • Το πλήκτρο Επιστροφή ενεργοποιεί την τρέχουσα επιλογή: η εντολή εκτελείται.

  • Το πλήκτρο Escape καθαρίζει την τρέχουσα επιλογή.

  • Το πλήκτρο Tab πραγματοποιεί εναλλαγή μεταξύ του αντικειμένου και του παραθύρου ενεργειών.

Η έννοια των πλήκτρων Space, ., ,, / και = δεν μπορεί να αλλάξει, αλλά μπορείτε να απενεργοποιήσετε την ειδική σημασία τους με το πλαίσιο ελέγχου Χρήση μονών εντολών πληκτρολόγησης στις Προτιμήσεις Kupfer.

Εμφάνιση βοήθειας

Πως να ρυθμίσετε μια συντόμευση πληκτρολογίου

  1. Ανοίξτε τις Προτιμήσεις Kupfer και πηγαίνετε στην καρτέλα Πληκτρολόγιο.

  2. Κάντε διπλό κλικ στη γραμμή της συντόμευσης Συντομεύσεις πληκτρολογίου περιηγητή.

  3. Χρησιμοποιώντας το παράθυρο που εμφανίζεται, πατήστε τα πλήκτρα τροποποίησης και ένα γράμμα ή αριθμό για να ορίσετε μια συντόμευση. Ένα παράδειγμα είναι πατώντας και κρατώντας πατημένο το Ctrl και, στη συνέχεια, πιέζοντας και αφήνοντας το T για να δεσμεύσετε τα Ctrl+T.

Action Accelerators

Action acclerators are a quick way to select or run a certain action on an object. All action accelerators use the same modifier key, which is Ctrl by default.

An action's accelerator is a single letter that is pressed together with the modifier to run the action (in the case of actions that need an indirect object, the third pane will be selected instead).

How to configure an action accelerator

  1. Open Kupfer and find an object that allows the action you are configuring. Switch to the action pane with Tab.

  2. Find the action to configure an accelerator for in the action pane.

  3. Click the kupfer context menu in the window and select the option Assign Accelerator to “(Action)”

  4. Type a single letter (without modifier) in the window that pops up. Escape cancels the configuration, and Backspace unsets the accelerator key for the action.

Regular keyboard shortcuts in Kupfer take precedence over action accelerator keys. If multiple simultaneously eligible actions have the same accelerator key, the highest ranked one will be activated.

Γενικές συντομεύσεις πληκτρολογίου

Το Kupfer ακούει πάντα στις γενικές συντομεύσεις, ακόμη και αν δεν είναι στο προσκήνιο.

Έχουν τις ακόλουθες σημασίες και προεπιλεγμένες συντομεύσεις:

Ctrl+Διάστημα

Εμφάνιση/απόκρυψη του Kupfer

(δεν έχει ρυθμιστεί από προεπιλογή)

Εμφάνιση του Kupfer με 'Επιλογή' εστιασμένο αντικείμενο

Πως να ρυθμίσετε μια γενική συντόμευση πληκτρολογίου

  1. Ανοίξτε τις Προτιμήσεις Kupfer και πηγαίνετε στην καρτέλα Πληκτρολόγιο.

  2. Κάντε διπλό κλικ στη γραμμή της συντόμευσης Γενικές συντομεύσεις πληκτρολογίου.

  3. Χρησιμοποιώντας το παράθυρο που εμφανίζεται, πατήστε τα πλήκτρα τροποποίησης και ένα γράμμα ή αριθμό για να ορίσετε μια γενική συντόμευση. Ένα παράδειγμα είναι πατώντας και κρατώντας πατημένο το Super και, στη συνέχεια, πιέζοντας και αφήνοντας το Space για να δεσμεύσετε τα Super+Space.

Δεδομένου ότι αυτές οι συντομεύσεις πληκτρολογίου είναι γενικές, μπορούν να χρησιμοποιηθούν από οποιαδήποτε εφαρμογή. Επιλέξτε συντομεύσεις με προσοχή έτσι ώστε να μην έρχονται σε αντίθεση με άλλες λειτουργίες.

Εναύσματα

Το πρόσθετο Εναύσματα σας επιτρέπει να ενεργοποιήσετε τις ενέργειες με τις γενικές συντομεύσεις πληκτρολογίου. Συντομεύσεις του Εναύσματα δεν εμφανίζονται στις Προτιμήσεις Kupfer.
