Calculateur

Le greffon « Calculateur »

Le greffon « Calculateur » vous permet de calculer rapidement des expressions. Il sait reconnaître les textes commençant par « = » comme une expression. Saisir « = » en ligne de commande démarre le mode texte avec le préfixe « = » pour un accès rapide.

Utilisation de base

  1. Activez Kupfer et appuyez sur =

  2. Saisissez une expression mathématique avec +,-,/,* (et ** pour l'exponentiation)

  3. Appuyez sur Entrée pour obtenir le résultat.

Notes

Le calculateur utilise les modules mathématiques et mathématiques complexes du langage Python et analyse les expressions comme étant des expressions Python. Vous pouvez utiliser les fonctions mathématiques standards, comme sqrt, sin, exp et log ; la commande =help affiche une liste de toutes les fonctions et de toutes les constantes disponibles.

  • Veuillez noter que l'opérateur de mise à la puissance en Python est deux étoiles ; par exemple, =3**3 donne 27.

  • Pour calculer les fonctions trigonométriques des angles, vous devez d'abord les convertir en radians :

    sin(radians(30)) -> 0.5

  • Le dernier résultat obtenu est enregistré dans la variable portant le nom _ (le caractère souligné, comme dans la console Python).

The Calculator lets you also calculate expression without "=" on the beginning as long it have inside one of operators: +, -, *, /, ^, |, &, ~.
