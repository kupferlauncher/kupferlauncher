Applications

Le greffon « Applications »

Applications donne accès à tous les programmes installés.

Utilisation de base

Lancement d'une application

  1. Activez Kupfer et saisissez les premières lettres du nom de l'application

  2. Appuyez sur Entrée pour lancer l'application.

Ouverture d'un fichier avec une application particulière

  1. Sélectionnez un fichier dans Kupfer

  2. Appuyez sur Tab et sélectionnez l'action Ouvrir avec...

  3. Appuyez sur Tab pour aller au troisième volet (objet indirect) et sélectionnez l'application qui servira à ouvrir le fichier.

L'action Ouvrir ouvre des fichiers et des dossiers avec l'application enregistrée par défaut pour ce type particulier de fichiers. Vous pouvez la configurer avec l'action Définir l'application par défaut....

Les associations de type de fichiers sont fournies par l'environnement du bureau et leur modification ici les modifie aussi sur le bureau. Cela n'est peut-être pas le cas si vous n'utilisez pas GNOME ou XFCE.

Sélection de l'application par défaut pour un type de fichiers ou de dossiers

  1. Sélectionnez un fichier ou un dossier dans Kupfer

  2. Appuyez sur Tab et sélectionnez l'action Définir l'application par défaut...

  3. Appuyez sur Tab pour aller au troisième volet (objet indirect) et sélectionnez l'application qui servira systématiquement à ouvrir ce type de fichiers.

Configuration

The configuration option Applications for Desktop Environment in the plugin's information pane in Kupfer Preferences determines how Kupfer should act with regard to the desktop environment configuration—certain applications request not to be shown depending on which desktop environment is used.

The default setting is blank which means that it will use the environment variable XDG_CURRENT_DESKTOP to identify the desktop environment automatically.
