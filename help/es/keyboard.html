En modo comando, las siguientes pulsaciones del teclado tienen un significado especial:

↓ o Espacio

↑ o Mayús+Espacio

Ir a la coincidencia anterior

→ o /

Retroceso

.

Activar el modo de texto libre

,

Poner un objeto seleccionado en la pila («truco de la coma»)

Teclas de acceso rápido:

=

Activar el modo de texto libre con el prefijo «=» (para la Calculadora)

/

Activar el modo de texto libre (cuando no hay nada seleccionado) con el prefijo / (para indicar una ruta desde la raíz)

Además:

  • Intro activa la selección actual: el comando se ejecuta.

  • Esc limpia la selección actual.

  • Tab cambia entre el objeto y el panel de acciones.

El significado de las teclas Espacio, ., ,, / y = no se puede cambiar, pero puede desactivar su significado especial con la casilla Usar comandos de una sola tecla en las Preferencias de Kupfer.

Cómo configurar un atajo de teclado

  1. Abra las Preferencias de Kupfer y vaya a la pestaña Teclado.

  2. Doble pulsación en la fila del atajo, en Atajos de teclado del examinador.

  3. Usando la ventana que aparece, pulse las teclas modificadoras y un número o una letra para establecer un atajo. Un ejemplo es pulsar y mantener Ctrl y entonces pulsar y soltar T para asociar Ctrl+T.

Action Accelerators

Action acclerators are a quick way to select or run a certain action on an object. All action accelerators use the same modifier key, which is Ctrl by default.

An action's accelerator is a single letter that is pressed together with the modifier to run the action (in the case of actions that need an indirect object, the third pane will be selected instead).

How to configure an action accelerator

  1. Open Kupfer and find an object that allows the action you are configuring. Switch to the action pane with Tab.

  2. Find the action to configure an accelerator for in the action pane.

  3. Click the kupfer context menu in the window and select the option Assign Accelerator to “(Action)”

  4. Type a single letter (without modifier) in the window that pops up. Escape cancels the configuration, and Backspace unsets the accelerator key for the action.

Regular keyboard shortcuts in Kupfer take precedence over action accelerator keys. If multiple simultaneously eligible actions have the same accelerator key, the highest ranked one will be activated.

Atajos de teclado globales

Kupfer siempre escucha los atajos globales, aunque actualmente no esté en primer plano.

Tienen el siguiente significado y los siguientes atajos predeterminados:

Ctrl+Espacio

Mostrar/ocultar Kupfer

(no configurado de manera predeterminada)

Mostrar Kupfer con el foco en el objeto «Selección».

Cómo configurar un atajo de teclado global

  1. Abra las Preferencias de Kupfer y vaya a la pestaña Teclado.

  2. Doble pulsación en la fila del atajo, en Atajos de teclado globales.

  3. Usando la ventana que aparece, pulse las teclas modificadoras y un número o una letra para establecer un atajo global. Un ejemplo es pulsar y mantener Super y entonces pulsar y soltar Espacio para asociar Super+Espacio.

Ya que estos atajos de teclado son globales, se puede usar desde cualquier aplicación. Seleccione los atajos con cuidado para que no entren en conflicto con otras funciones.

Disparadores

El complemento Disparadores permite activar acciones con atajos de teclado globales. Los atajos del disparador no aparecen en las Preferencias de Kupfer.

Consulte la Disparadores para obtener más información.
