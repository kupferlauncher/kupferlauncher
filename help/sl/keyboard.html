Kako nastaviti tipkovno bližnjico

  1. Odprite Možnosti Kupfer in pojdite v zavihek Tipkovnica.

  2. Dvokliknite na vrstico bližnjice pod Tipkovne bližnjice brskalnika.

  3. V oknu, ki se pojavi, za nastavitev splošne bližnjice pritisnite na spremenilne tipke in črko ali število. Primer je pritisk in držanje na tipko Ctrl in nato pritisk in izpust tipke T za vezavo Ctrl+T.

Action Accelerators

Action acclerators are a quick way to select or run a certain action on an object. All action accelerators use the same modifier key, which is Ctrl by default.

An action's accelerator is a single letter that is pressed together with the modifier to run the action (in the case of actions that need an indirect object, the third pane will be selected instead).

How to configure an action accelerator

  1. Open Kupfer and find an object that allows the action you are configuring. Switch to the action pane with Tab.

  2. Find the action to configure an accelerator for in the action pane.

  3. Click the kupfer context menu in the window and select the option Assign Accelerator to “(Action)”

  4. Type a single letter (without modifier) in the window that pops up. Escape cancels the configuration, and Backspace unsets the accelerator key for the action.

Regular keyboard shortcuts in Kupfer take precedence over action accelerator keys. If multiple simultaneously eligible actions have the same accelerator key, the highest ranked one will be activated.

Splošne tipkovne bližnjice

Kupfer vedno posluša za splošne bližnjice, tudi če ni v ospredju.

Imajo naslednje pomene in privzete bližnjice:

Ctrl+Preslednica

Pokaže/skrije Kupfer.

(ni privzeto nastavljeno)

Pokaže Kupfer s predmetom 'Izbira' v žarišču

Kako nastaviti splošno tipkovno bližnjico

  1. Odprite Možnosti Kupfer in pojdite v zavihek Tipkovnica.

  2. Dvokliknite na vrstico bližnjice pod Splošne tipkovne bližnjice.

  3. V oknu, ki se pojavi, za nastavitev splošne bližnjice pritisnite na spremenilne tipke in črko ali število. Primer je pritisk in držanje na tipko Super in nato pritisk in izpust tipke Preslednica za vezavo Super+Preslednica.

Ker so te tipkovne bližnjice splošne, jih lahko uporabite v kateremkoli programu. Bližnjice izberite previdno, da ne bodo v sporu z drugimi zmožnostmi.
