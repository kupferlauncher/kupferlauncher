Anwendungen

Anwendungen-Plugin

Anwendungen stellt alle installierten Programme bereit.

Grundlagen der Bedienung

Starten einer Anwendung

  1. Aktivieren Sie Kupfer und geben Sie eine Abkürzung des Anwendungsnamens ein.

  2. Drücken Sie die Eingabetaste, um die Anwendung zu starten.

Eine Datei mit einer bestimmten Anwendung öffnen

  1. Eine Datei in Kupfer auswählen

  2. Type Tab and select the action Open With...

  3. Type Tab to select the third (indirect object) pane and in it, select the application to open the file with.

The action Open opens files and folders with the registered default application for that specific file type. This can be configured using the action Set Default Application....

The file type associations are provided by the desktop environment, and modifying them here will modify them on the desktop as well. This may not be true when not using GNOME or XFCE.

Auswahl der Standardanwendung für einen Dateityp und für Ordner

  1. Dateien oder Ordner in Kupfer auswählen

  2. Type Tab and select the action Set Default Application...

  3. Type Tab to select the third (indirect object) pane and in it, select the application to always open the file type with.

Konfiguration

The configuration option Applications for Desktop Environment in the plugin's information pane in Kupfer Preferences determines how Kupfer should act with regard to the desktop environment configuration—certain applications request not to be shown depending on which desktop environment is used. By default, it behaves like GNOME.
