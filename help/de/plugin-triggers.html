Auslöser

Auslöser-Plugin

Mit Auslösern können Sie einen Befehl angeben, den Sie normalerweise in Kupfer ausführen, und diesen Befehl einer globalen Tastenkombination zuordnen. Beispiele dafür sind das Starten einer Anwendung oder das Öffnen eines Dokuments. Während Kupfer läuft, kann jede Anwendung aktuell fokussiert sein.

Grundlagen der Bedienung

Anlegen eines neuen Auslösers

  1. Geben Sie in Kupfer einen Befehl an, den Sie ausführen wollen.

  2. Drücken Sie Strg+Eingabetaste, um einen zusammengesetzten Befehl zu erstellen.

  3. Drücken Sie die Tabulatortaste, wählen Sie die Aktion Auslöser hinzufügen … und drücken Sie die Eingabetaste.

  4. Drücken Sie nun Modifikationstasten und eine Buchstaben- oder Zifferntaste, um ein globales Tastenkürzel festzulegen. Beispiel: Drücken Sie Strg+Alt und halten Sie diese Tasten gedrückt. Dann drücken Sie R und lassen die Taste wieder los, um die Tastenkombination Strg+Alt+R zu verwenden.

Entfernen eines Auslösers

  1. Schauen Sie in Kupfer nach den festgelegten Auslösern und drücken Sie →, um deren Inhalt anzusehen.

  2. Suchen Sie nach dem aktiven Auslöser in der Liste und wählen Sie Auslöser entfernen.

Da die Auslöser-Tastenkombinationen globaler Natur sind, können sie aus jeder Anwendung heraus aufgerufen werden. Wählen Sie die Tastenkombinationen mit Vorsicht, damit sie nicht mit anderen Funktionen in Konflikt geraten.

Profitipps

Proxy-Objekte wie Ausgewählter Text, Ausgewählte Datei, Vorderstes Fenster, Letzter Befehl usw. ermöglichen sehr mächtige Auslöser.

Die Anwendungsaktion Starten ist fallabhängig. Wenn Sie einen Auslöser an Starten binden, dann wird die Anwendung gestartet, sofern sie noch nicht läuft. Sollte sie jedoch schon laufen, erhält sie den Fokus. Im Gegensatz dazu startet die Aktion Erneut starten die Anwendung, ganz gleich, ob sie schon läuft oder nicht.

Beispiele für Auslöser

Texteditor → starten

Starten den Texteditor oder fokussiert dessen Fenster, falls er schon läuft.

Ausgewählter Text → Suchen mit → Google

Sucht im Web nach dem aktuell ausgewählten Text.

Ausgewählte Datei → Verschieben nach → (Download-Ordner)

Verschiebt die aktuell ausgewählte Datei in einem bestimmten Ordner

Musiktitel → Inhalte durchsuchen

Sie können einen Auslöser festlegen, um in Kupfer einen bestimmten unterkatalog zu öffnen, zum Beispiel die Titel in Rhythmbox.
